Estrutura de Grafos em C++

O problema consiste na formação de N grupos a partir de um conjunto de M pessoas, utilizando conceitos de grafos implementados em C++.

Cada pessoa possui as seguintes características:

- Coeficiente de eficiência, que representa o desempenho obtido quando trabalha em conjunto com determinados colegas de equipe;
- Salário, que indica o custo individual.

O objetivo é construir os N grupos respeitando as seguintes restrições:

- O salário total de cada grupo não deve ultrapassar um limite pré-estabelecido;
- Todos os grupos devem apresentar valores de eficiência semelhantes, garantindo equilíbrio de desempenho;
- Todos os grupos devem possuir quantidades semelhantes de pessoas, evitando grandes discrepâncias entre eles.
